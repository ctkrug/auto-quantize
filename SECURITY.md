# Security Policy

## Reporting a vulnerability

If you find a security issue in auto-quantize (e.g. unsafe handling of
downloaded files, path traversal in output paths, or an issue in how HTTP
responses are parsed), please open a private report via GitHub's
[Security Advisories](https://github.com/ctkrug/auto-quantize/security/advisories/new)
for this repository instead of a public issue.

## Supported versions

This project is pre-1.0; only the latest commit on `main` is supported.
